# Try ProtoAgent

Experience ProtoAgent directly in your browser. This terminal connects to a Cloudflare Worker running Workers AI.

<Terminal />

## Getting Started

1. **Wait for the terminal to load** - it will automatically create a session
2. **Type your message** and press Enter
3. **Watch it work** - ProtoAgent will use tools to help you

## What You Can Do

- **Read files**: "Show me the README"
- **Write code**: "Create a hello world function"
- **Search**: "Find TODO comments in src/"
- **Manage tasks**: "Add 'fix bug' to the todo list"

## Limitations

This demo runs in a sandboxed environment:
- Files are stored in memory (not persisted)
- Daily message quota applies
- No shell command execution

For the full experience, [install ProtoAgent locally](/try-it-out/getting-started).
